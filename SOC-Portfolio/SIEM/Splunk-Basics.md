# Splunk Basics

## What is a SIEM?
A SIEM (Security Information and Event Management) system collects,
stores, and analyzes logs from multiple sources to help detect,
investigate, and respond to security incidents.

## What is Splunk used for in a SOC?
Splunk is used by SOC analysts to search logs, investigate alerts,
identify suspicious activity, and support incident response.

## What is an index in Splunk?
An index is where Splunk stores and organizes ingested log data,
allowing analysts to efficiently search and analyze events.

## How does Splunk help SOC analysts?
Splunk enables correlation of events from different systems,
making it easier to detect attacks, investigate incidents,
and reduce false positives.
